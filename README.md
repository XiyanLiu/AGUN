# AGUN
Geometric Rectification of Document Images using Adversarial Gated Unwarping Network  
## Requirements  
Python 3.6  
Pytorch 0.4.1  
Opencv  
## Folder  
* code: the code of data synthesis.  
* dataset: contains DOC_SYN dataset, OCR_REAL dataset and OCR_SYN dataset.
